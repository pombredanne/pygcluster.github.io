resample(distance=None, linkage=None, iterations=10000, minClusterSize=4, minClusterFrequency=0.001, picklePrefix=None)¶

inject noise & exec hierarchical clustering & filter 0.1 % of threshold

Currently, the original data is never used for clustering, but immediately altered via gauss-distribution.

  1. alter data according to stdev
  2. calculate linkage matrix (via hierarchy.linkage)
  3. translate linkage matrix and store clusters in array Data
Parameters:distance (string) – distance method, e.g. ‘euclidean’

  • topX (float) – get the top X clusters (sorted by frequency)

Note

frequencies(identifier=None, index=None, cluster=None)¶

Returns the frequencies for a given identifier (e.g. gene or FastaID) as dictionary or None if identifier is not part of the data set, index out of range or cluster was not found during iteration.

Example

>>> cluster.frequencies( identifier = 'jgi|Chlre4|513072|au5.g3286_t1' )
... defaultdict(<type 'float'>, {'average-correlation': 0.0, 'complete-correlation': 0.0, 
... 'centroid-euclidean': 0.0015, 'median-euclidean': 0.0064666666666666666, 
... 'ward-euclidean': 0.0041333333333333335, 'weighted-correlation': 0.0, 
... 'complete-euclidean': 0.0014, 'weighted-euclidean': 0.0066333333333333331, 
... 'average-euclidean': 0.0020333333333333332})
Parameters:
  • identifier (string) – search frequencies by identifier input
  • index (int) – search frequencies by identifier index input
  • cluster (tuple of ints) – search frequencies by cluster input
